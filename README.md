# terraform-portfolio-project
This portfolio holds code, files and content for a Next.js portfolio website hosted on AWS. The static web site content is held in Amazon S3, and Amazon CloudFront is applied as the CDN.

<h1>Project Brief</h1>

<p>Client: Jade Smith, freelance web designer has built a modern, responsive single-page portfolio website on Next.js. She needs this hosted globally as she is seeking clients worldwide. The site needs to be highly available and accessible worldwide with minimal downtime.</p>

<h2>Other important requirements:</h2>
<ul>
<li>It needs to be scalable, and should handle increasing traffic without performance degradation.</li>
<li>Must be cost-Effective: S3 can provide optimized hosting costs without unnecessary expenses.</li>
<li>Must be fast Loading: Quick loading times for all visitors globally</li>
</ul>


<h2>Steps needed to build and deploy</h2>
<ul>
<li>Build Next.js templates to export a static web page</li>
<li>Robust: S3</li>
<li>Scalable: S3</l1>
<li>Cost-effective: S3</li>
<li>Globally available: Cloudfront</l1>
<li>Fast loading times: CloudFront</li>
<li>Fast deployment of Portfolio Website to AWS: IaC Terraform infrastructure build</l1>
</ul>
